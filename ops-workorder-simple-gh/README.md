# 运维工单系统 · Simplified Edition (GitHub @glitteringuniverse)

本仓库基于 GitHub 账户 **glitteringuniverse**，已将 Maven groupId 与 Java 包路径替换为 `io.github.glitteringuniverse`。

快速启动
```bash
cd deploy && ./run-dev.sh   # 启动数据库/缓存/消息队列 + 后端
cd frontend-web && pnpm i && pnpm dev   # 启动前端 (http://localhost:5173)
```
