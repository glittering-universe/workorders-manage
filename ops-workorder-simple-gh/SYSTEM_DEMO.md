# 运维工单管理系统演示指南

## 🎯 系统概述

这是一个完整的企业级运维工单管理系统，具备以下核心功能：

### ✨ 主要特性
- 🔐 用户认证与角色权限管理
- 📋 工单全生命周期管理
- ✅ 多级审批流程
- 👥 用户和部门管理
- 📊 实时统计分析
- 📱 响应式界面设计

## 🚀 快速体验

### 1. 系统启动
```bash
# 启动后端服务 (端口 8080)
cd backend && mvn spring-boot:run

# 启动前端服务 (端口 5174)
cd frontend-web && npm run dev
```

### 2. 访问系统
- 前端地址：http://localhost:5174
- 后端API：http://localhost:8080/api

### 3. 测试账号
| 角色 | 用户名 | 密码 | 权限描述 |
|------|--------|------|----------|
| 系统管理员 | admin | password123 | 全系统管理权限 |
| 部门经理 | manager1 | password123 | 部门管理、审批权限 |
| 运维人员 | operator1 | password123 | 工单执行权限 |
| 普通用户 | user1 | password123 | 工单创建权限 |

## 📋 完整工作流演示

### 场景1：硬件故障处理流程
1. **普通用户创建故障工单**
   - 登录：user1 / password123
   - 创建工单：服务器硬盘故障
   - 状态：草稿 → 提交

2. **部门经理审批**
   - 登录：manager1 / password123
   - 审批中心：查看待审批工单
   - 操作：批准工单

3. **工单分派**
   - 管理员分派给运维人员
   - 状态：已审批 → 已分派

4. **运维执行**
   - 登录：operator1 / password123
   - 执行工单：更换硬盘
   - 状态：执行中 → 已完成

### 场景2：系统变更管理
1. **变更申请**：网络设备升级
2. **风险评估**：高优先级变更
3. **审批流程**：多级审批
4. **实施计划**：分派专业人员
5. **变更执行**：按计划实施
6. **结果验证**：完成确认

## 🔧 技术架构

### 后端技术栈
- **Spring Boot 3.3.0** - 应用框架
- **Spring Data JPA** - 数据访问层
- **MySQL 8.4** - 关系型数据库
- **Flyway** - 数据库版本管理
- **BCrypt** - 密码加密

### 前端技术栈
- **Vue.js 3** - 响应式框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Axios** - HTTP客户端

### 数据库设计
- `users` - 用户信息表
- `work_order` - 工单主表
- `work_order_log` - 操作日志表
- `approval_flow` - 审批流程表

## 🧪 API测试命令

### 用户认证
```bash
# 用户登录
curl -X POST http://localhost:8080/api/users/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"password123"}'
```

### 工单操作
```bash
# 获取所有工单
curl http://localhost:8080/api/orders

# 创建工单
curl -X POST "http://localhost:8080/api/orders?creatorId=1" \
  -H "Content-Type: application/json" \
  -d '{"title":"测试工单","category":"MAINTENANCE","priority":"MEDIUM","slaMinutes":240}'

# 提交工单
curl -X POST "http://localhost:8080/api/orders/1/submit?userId=1"

# 审批工单
curl -X POST "http://localhost:8080/api/orders/1/approve?approverId=2" \
  -H "Content-Type: application/json" \
  -d '{"comment":"审批通过"}'
```

### 统计查询
```bash
# 获取日统计
curl "http://localhost:8080/api/statistics/daily?date=2025-06-08"
```
